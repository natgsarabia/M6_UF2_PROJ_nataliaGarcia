¡Bienvenido a mi web!

Para esta web he utilizado tres APIs web, que serían:

  • Web Notification API: Para avisar al usuario si se encuentrar restaurantes en la población escogida.
  • Web Share API: Para compartir, si se quiere, la información de un restaurante. (el restaurante debe de tener url, si no envía el URL de localhost :( ).
  • Web Animations API: Para añadir movimiento a la apertura y cierre de modales.
